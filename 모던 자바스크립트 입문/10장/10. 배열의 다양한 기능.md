### 10.1.2 배열 수정 메서드

```js
var a = ["A", "B", "C"];

a.push("D") // 배열의 길이 반환
a.pop() // 마지막 값 반환
a.shift(); // 첫 번째 요소 반환 후 제거
a.unshift(); // 첫 번째 요소에 추가, 배열의 길이 반환
a.splice()
a.slice
a.sort
```
